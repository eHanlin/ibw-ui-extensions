ibw-ui-extensions
======================================


## Install

```sh
npm install --save ibw-ui-extensions
```
or

```sh
bower install ibw-ui-extensions
```

## Usage

```js
angular.module('module', ['ibw.ui.extensions']);
```

```html
<ib-item-content ib-item-content="item" on-answer-updated="onAnswerUpdated(result)"></ib-item-content>
```

## Run

```sh
gulp server
```

## Build

```sh
gulp
```
